# 付诸实践

正如本指南所涵盖的，在交付能力和遵循最佳实践方面有许多细微差别。

成功有四个关键支柱：

1. 在注册过程中设置适当的期望值，并有一个可防止地址错误的良好注册过程。
2. 提供相关、及时的内容。
3. 通过删除变坏的地址来维护您的列表。
4. 随时随地进行监视、测试和调整。

如果您在问题上不清楚或需要帮助，请联系您的Adobe可交付性顾问或可交付性专家寻求帮助。
