# 持续监测

下面是一些帮助确定您可能需要专家支持的可能问题的方法：

* 观察到硬跳出或软跳出次数激增。 这可能表明存在阻止列表或其他可投放性问题。
* 在投放率保持较高的情况下，打开和点击量度显着减少。 这指示了潜在的批量文件夹放置。
* 投诉显着增加。 这可能是由于列表源质量不佳导致的。
* 您有可能会影响可投放性的任何策略计划。 这些包括但不限于订阅者获取、参与策略、季节性策略或频率的显着变化，以及营销活动类型。
