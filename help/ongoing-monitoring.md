# 持续监测

以下是帮助您确定可能需要专家支持的问题的一些方法：

* 在硬或软弹跳中有观察到的尖峰。 这可能表明存在阻塞性清单或其他交付问题。
* 在交付率保持较高的同时，打开和点击量度显着减少。 这表示可能放置批量文件夹。
* 投诉大幅增加。 这可能是由质量差的列表源造成的。
* 您拥有任何可能影响交付能力的战略计划。 这包括但不限于用户赢取率、互动策略、季节性策略或频率和活动类型的显着变化。
